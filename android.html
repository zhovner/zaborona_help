Инструкция по настройке Zaborona VPN для Android
Инструкция подходит для устройств с версией Android ≥ 4.0
1
Установите приложение OpenVPN клиент из PlayMarket
https://play.google.com/store/apps/details?id=net.openvpn.openvpn
2
Скачайте файл конфигурации Zaborona VPN:
https://zaborona.help/zaborona-help.ovpn
3
Запустите файл конфигурации и импортируйте его в программу OpenVPN.
Подключитесь к VPN.
После подключения в области уведомления появится значок OpenVPN.

VPN активируется только для заблокированных сайтов и приложений. Список заблокированных сетей можно посмотреть здесь. Остальные сайты будут работать как и раньше напрямую через вашего провайдера.
