Инструкция по настройке Zaborona VPN для iOS
Данная инструкция работает на iOS ≥ 8
1
Установите приложение OpenVPN Connect из AppStore.
2
Скачайте профиль автоконфигурации VPN.
Подключение к серверам Zaborona.help через Европу (у кого не работает прямое подключение):
srv0.zaborona-help_maxroutes.ovpn TCP
srv0.zaborona-help-UDP-no-encryption_maxroutes.ovpn UDP
(расширенный Европа (тестовый), для доступа к Mail.ru, ОК, ВК, Касперский, Dr. Web, WebMoney и другим сайтам). Данная конфигурация рассчитана больше для тех, у кого не работает прямое подключение к серверам Zaborona.help.
Важно: Используйте браузер Safari. При открытии ссылки через мессенджеры или из почты файл автоконфигурации может не работать.
3
Импортируйте файл автоконфигурации и подключитесь к VPN.
После подключения вы увидите значок VPN в области уведомлений вверху. Для управления подключением используйте программу OpenVPN Connect.

VPN активируется только для заблокированных сайтов и приложений. Список заблокированных сетей можно посмотреть здесь. Остальные сайты будут работать, как и раньше, напрямую через вашего провайдера.

Постоянное подключение к VPN может незначительно влиять на расход батареи.
Как удалить Zaborona VPN с устройства iOS/MacOS
Для полного удаления Zaborona VPN с устройства удалите приложение OpenVPN Connect.
