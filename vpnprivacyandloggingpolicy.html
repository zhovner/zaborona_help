Политика конфиденциальности и журналирования VPN-сервиса
Последнее обновление: 1 января 2025 г.
Настоящая Политика описывает порядок обработки технической информации при использовании VPN-сервиса Zaborona.help.
Данный документ распространяется исключительно на работу VPN-сервиса и не относится к веб-сайту Zaborona.help. Порядок обработки информации при посещении сайта описан в документе «Политика конфиденциальности».

1. Принципы обработки данных
При разработке и эксплуатации VPN-сервиса мы придерживаемся следующих принципов:
- сбор только минимально необходимой технической информации;
- минимизация объёма сохраняемых данных;
- отсутствие анализа содержимого пользовательского трафика;
- защита конфиденциальности пользователей;
- обеспечение стабильной и безопасной работы инфраструктуры.
2. Какие данные необходимы для работы VPN
Для установления VPN-соединения сервер может временно обрабатывать техническую информацию, необходимую для обеспечения работы сервиса, включая:
- IP-адрес пользователя;
- время установления и завершения соединения;
- объём переданных данных (при необходимости для работы отдельных компонентов сервиса);
- DNS-запросы в незашифрованном виде, которые передаются через VPN-соединение и при этом не связываются с конкретными пользователями;
- технические параметры VPN-соединения.

Обработка данной информации осуществляется исключительно в целях функционирования сервиса.
3. Обработка DNS-запросов
Для обеспечения корректной работы VPN-сервиса может выполняться временная обработка DNS-запросов.
Такая обработка осуществляется исключительно в технических целях, необходимых для маршрутизации трафика, диагностики и обеспечения стабильной работы сервиса.
DNS-запросы не используются для анализа пользовательской активности и не связываются с конкретными пользователями. При обработке применяется обезличивание технической информации, а данные хранятся только в течение минимального времени, необходимого для функционирования сервиса, после чего автоматически удаляются.
4. Журналирование (Logging)
Сервис не ведёт журналов пользовательской активности. Для обеспечения стабильной работы инфраструктуры могут временно создаваться технические журналы, не содержащие информации о действиях пользователя в сети Интернет.

В частности, сервис не сохраняет:
- историю посещённых веб-сайтов;
- содержимое интернет-трафика;
- данные о посещаемых интернет-ресурсах;
5. Технические журналы
Для обеспечения стабильной работы инфраструктуры отдельные серверы могут временно создавать технические журналы, содержащие информацию, необходимую для:
- диагностики неисправностей;
- обеспечения безопасности инфраструктуры;
- предотвращения злоупотреблений;
- устранения технических ошибок.

Такие журналы не предназначены для отслеживания пользовательской активности и используются исключительно в технических целях.
6. Что мы не собираем
VPN-сервис не предназначен для отслеживания действий пользователей в сети Интернет.
Мы не анализируем содержимое передаваемого трафика и не формируем профили пользователей на основании их интернет-активности.
Техническая информация, временно обрабатываемая для обеспечения работы сервиса, не используется для идентификации пользователей или анализа истории их посещений.
7. Защита информации
Мы принимаем разумные технические и организационные меры для защиты инфраструктуры VPN-сервиса и предотвращения несанкционированного доступа к технической информации.
8. Передача информации третьим лицам
Мы не продаём и не передаём информацию о пользователях третьим лицам.
Информация может быть раскрыта исключительно в случаях, прямо предусмотренных текущим законодательством.
9. Изменения настоящей Политики
Настоящая Политика может периодически обновляться.
Актуальная версия всегда публикуется на сайте Zaborona.help.
10. Контакты
По вопросам, связанным с настоящей Политикой, вы можете связаться с администрацией проекта через контактные данные, указанные на сайте.
