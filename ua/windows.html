Zaborona VPN для windows
Рекомендується використовувати Windows 10/11
1
Завантажте OpenVPN Community-клієнт для Windows 7/8/8.1/10/11. Можна також використовувати версію OpenVPN Connect.
2
Завантажте файл конфігурації Zaborona VPN:
srv0.zaborona-help-TCP_maxroutes.ovpn TCP
srv0.zaborona-help-UDP_maxroutes.ovpn UDP

3
Запустіть OpenVPN. Натисніть правою кнопкою миші на OpenVPN у треї (значок комп'ютера внизу праворуч, біля годинника). Натисніть "Імпорт конфігурації" та виберіть завантажений файл zaborona-help.ovpn.
4
Натисніть правою кнопкою на OpenVPN значок, встановіть з'єднання.

Зверніть увагу, що остання версія OpenVPN, яка підтримує Windows XP, - це 2.3.18, яку можна завантажити як у 32-разрядній, так і в 64-разрядній версії, яку можна завантажити за посиланням тут.
